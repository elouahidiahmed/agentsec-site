Suppression des données

Dernière mise à jour : 21 juin 2026

Cette page explique quelles données sont associées à nos applications et jeux, et comment en demander la suppression.

Pas de compte : nos applications ne créent pas de compte utilisateur et ne collectent aucune information personnelle identifiable. Les seules données traitées sont des données de mesure d'audience et de diagnostic anonymisées (voir notre politique de confidentialité).

1. Données concernées

  • Données d'usage anonymisées (événements, écrans, version de l'app) ;
  • Données techniques de l'appareil (modèle, version d'Android) ;
  • Rapports de plantage et journaux de diagnostic ;
  • Identifiant d'installation/d'instance généré par les outils de mesure.

Nous ne détenons aucune donnée permettant de vous identifier directement (nom, e-mail, adresse, etc.).

2. Demander la suppression

Pour demander la suppression des données associées à votre appareil :

  1. Envoyez un courriel à contact@agentsec.ca avec l'objet « Suppression de données ».
  2. Indiquez le nom de l'application/jeu concerné.
  3. Pour nous permettre de cibler les données, fournissez si possible votre identifiant d'instance d'application (App Instance ID) ou, à défaut, le modèle de l'appareil et la période d'utilisation approximative.

Nous traitons les demandes et procédons à la suppression auprès de nos fournisseurs (Firebase) dans un délai maximal de 30 jours.

3. Supprimer vous-même les données locales

Vous pouvez aussi : désinstaller l'application (ce qui supprime les données locales), ou effacer les données de l'app via Paramètres → Applications → [l'app] → Stockage → Effacer les données. Vous pouvez également réinitialiser votre identifiant publicitaire dans Paramètres → Confidentialité → Annonces.

4. Conservation

En l'absence de demande, les données anonymisées sont automatiquement supprimées au terme des durées de rétention de nos fournisseurs (à titre indicatif, jusqu'à 14 mois pour l'usage et environ 90 jours pour les rapports de plantage).
