Support

Assistance pour nos applications et jeux

Une question, un bug, une suggestion ? Nous sommes là pour aider.

Nous écrire

Contactez-nous par courriel à contact@agentsec.ca. Nous nous efforçons de répondre sous 2 jours ouvrables.

Pour un traitement plus rapide

Lorsque vous nous écrivez au sujet d'un problème, merci d'inclure si possible :

  • le nom de l'application ou du jeu concerné ;
  • le modèle de votre appareil et la version d'Android ;
  • la version de l'application (visible dans la fiche Google Play) ;
  • une description du problème et, si possible, une capture d'écran ;
  • les étapes pour reproduire le problème.

Confidentialité et données

Consultez notre politique de confidentialité. Pour demander la suppression de vos données, rendez-vous sur la page Suppression des données.
