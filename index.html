Analyse de malware
conçue pour les agents IA

Un agent soumet un fichier, paie à l'analyse via x402 / HTTP 402, et reçoit un verdict structuré : le « VirusTotal » des agents. Cœur d'analyse : Assemblyline 4.

Voir l'API Nous contacter
🤖

Pensé pour les agents

Livraison MCP + REST. Un agent négocie le paiement et récupère un verdict machine-readable, sans humain dans la boucle.

💸

Paiement x402

Défi HTTP 402 conforme, règlement USDC via facilitateur, anti-rejeu, screening de sanctions, crédit si l'analyse échoue.

🧬

Verdict structuré

Contrat versionné report.v1 + STIX. Adressage par sha256 : dédup, idempotence et cache de coût.

📊

Observable

Orchestration 2-tiers (statique synchrone, deep asynchrone), métriques Prometheus et dashboard Grafana de pilotage.

Soumettre un fichier en une requête

# Tier STATIC — synchrone, renvoie un rapport immédiat
curl -F "file=@sample.bin" -F "tier=STATIC" \
     https://agentsec.ca/v1/analyses

# Verdict en cache (le chemin le moins cher)
curl https://agentsec.ca/v1/files/<sha256>/verdict
